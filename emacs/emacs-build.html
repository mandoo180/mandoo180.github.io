2. Quick Start

git clone https://github.com/emacs-mirror/emacs.git
cd emacs

./autogen.sh                                    # Generate configure script
./configure                                             # Detect system configuration
make                                                    # Build emacs
src/emacs                                               # Test emacs executable file
make install                                    # Install emacs globally
