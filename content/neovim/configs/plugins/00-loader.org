* Plugin Categories

** Core Dependencies
- **plenary.nvim**: Lua functions library used by many plugins
- **nui.nvim**: UI component library

** File Management & Navigation
- **telescope.nvim**: Fuzzy finder for files, buffers, and more
- **neo-tree.nvim**: File explorer tree
- **nvim-web-devicons**: File type icons

** UI Enhancements
- **bufferline.nvim**: Enhanced buffer/tab line
- **lualine.nvim**: Statusline plugin
- **onedark.nvim**: OneDark color scheme

** Debugging
- **nvim-dap**: Debug Adapter Protocol client
- **nvim-dap-ui**: UI for nvim-dap
- **nvim-dap-virtual-text**: Virtual text support for debugging
- **nvim-nio**: Async I/O library

** Syntax & Editing
- **nvim-treesitter**: Treesitter configurations
- **nvim-treesitter-textobjects**: Additional text objects based on treesitter
- **mini.pairs**: Auto pairs for brackets and quotes
- **emmet-vim**: Emmet support for web development

** Motion & Navigation
- **leap.nvim**: General-purpose motion plugin
- **vim-repeat**: Enable repeating plugin maps

** Completion
- **nvim-cmp**: Completion plugin
- **cmp-nvim-lsp**: LSP source for nvim-cmp
- **cmp-buffer**: Buffer source for nvim-cmp
- **cmp-path**: Path source for nvim-cmp
- **LuaSnip**: Snippet engine
- **cmp_luasnip**: LuaSnip source for nvim-cmp
- **friendly-snippets**: Collection of snippets

** Language Server Protocol
- **nvim-lspconfig**: Quickstart configurations for LSP
- **mason.nvim**: Portable package manager for LSP servers
- **mason-lspconfig.nvim**: Bridge between mason and lspconfig
- **nvim-jdtls**: Extensions for Java development
