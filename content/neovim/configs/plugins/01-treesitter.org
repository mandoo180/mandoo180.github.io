* Overview

Treesitter provides advanced syntax highlighting, indentation, and text objects based on tree-sitter parsers.

* Configuration

* Features Explained

** Ensure Installed
A comprehensive list of tree-sitter parsers to be installed, covering:
- Programming languages (C, Python, Ruby, Rust, Go, Java)
- Web technologies (HTML, CSS, JavaScript, TypeScript)
- Configuration formats (YAML, TOML, JSON, INI)
- Documentation formats (Markdown, Vimdoc)
- Version control (Git commit, Git config, Gitignore)
- Infrastructure (Dockerfile, Nginx, SSH config)

** Highlight Configuration
- Enabled by default for syntax highlighting
- Automatically disabled for large files (>100KB) to prevent performance issues
- No additional vim regex highlighting (using pure treesitter)

** Indentation
Treesitter-based indentation is enabled for more accurate auto-indenting.

** Text Objects
Custom text objects for better code navigation:
- =af= - Around function
- =if= - Inside function  
- =ac= - Around class
- =ic= - Inside class

With =lookahead= enabled, the cursor will automatically jump forward to find the text object.
