* How to run

*** Build html files

#+begin_src shell
  ./build.sh
#+end_src

*** Run server in Emacs

Install =simple-httpd=.\\
#+begin_src emacs-lisp
  (package-install 'simple-httpd)
#+end_src

*** Serve a Directory

Serve `./public' by =M-x httpd-serve-directory=.
